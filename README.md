## General info
A Django blogging app It has [features](#features) of a standard blogging platform.


## Features

* User authentication 
* User Login
* Author Password Reset
* Category List
* Category Articles List
* Post Like
* Post editing and deleting for Author
* Comments for Posts
* Articles Search
* Pagination
* Responsive


## Technologies
* Python 3
* Django 4
* HTML5
* CSS3
* Javascript
* Jquery 
* Ajax
* Font awesome
* CKEditor
* SQLite
